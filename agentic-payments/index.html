Build this into your app

Real endpoints and code you can copy. No placeholders — these work against https://api.affix-io.com.

  • 1. Get an API key: sign up at api.affix-io.com/docs or use POST /api/demo-key for testing (3 req/min per IP).
  • 2. Generate a proof with POST /api/proof/generate (userId, rules, data.authorized, transactionAmount).
  • 3. Submit to POS with POST /api/pos/submit-proof (proof, publicInputs, transactionId, amount, currency).
  • Production: Use the affixiomerchant SDK for offline-first agentic payments (Node, browser, Deno).
Terminal — replace YOUR_KEY with key from POST /api/demo-key
# 1) Get demo key
curl -s -X POST https://api.affix-io.com/api/demo-key -H "Content-Type: application/json" -d '{}'

# 2) Generate proof (use .key from step 1)
curl -s -X POST https://api.affix-io.com/api/proof/generate \
  -H "Content-Type: application/json" -H "Authorization: Bearer YOUR_KEY" \
  -d '{"userId":"agent_1","rules":["payment_authorized","amount_valid"],"data":{"authorized":true,"amountValid":1},"transactionAmount":19.99}'

# 3) Submit as payment (paste .proof and .publicInputs from step 2)
curl -s -X POST https://api.affix-io.com/api/pos/submit-proof \
  -H "Content-Type: application/json" -H "Authorization: Bearer YOUR_KEY" \
  -d '{"proof":"","publicInputs":{"decision":1,"verified":true,"timestamp":"...","constraints":["payment_authorized"],"amount":19.99},"transactionId":"txn_001","amount":19.99,"currency":"GBP"}'
Browser (same origin or CORS-allowed)
const API = 'https://api.affix-io.com';

async function runAgenticPayment() {
  const { key } = await fetch(API + '/api/demo-key', { method: 'POST', headers: { 'Content-Type': 'application/json' } }).then(r => r.json());
  const gen = await fetch(API + '/api/proof/generate', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json', 'Authorization': 'Bearer ' + key },
    body: JSON.stringify({
      userId: 'agent_1',
      rules: ['payment_authorized', 'amount_valid'],
      data: { authorized: true, amountValid: 1 },
      transactionAmount: 19.99
    })
  }).then(r => r.json());
  const pos = await fetch(API + '/api/pos/submit-proof', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json', 'Authorization': 'Bearer ' + key },
    body: JSON.stringify({
      proof: gen.proof,
      publicInputs: gen.publicInputs,
      transactionId: 'txn_' + Date.now(),
      amount: 19.99,
      currency: 'GBP'
    })
  }).then(r => r.json());
  return pos; // { success: true, transaction: { ... } }
}

Real response shapes

  • POST /api/demo-key → { "key": "affix_...", "keyPrefix": "affix_...", "warning": "...", "rateLimit": { "perIP": 3, "windowSeconds": 60 } }
  • POST /api/proof/generate → { "proofId", "proof", "publicInputs", "timestamp", "sector", "proofSize" }
  • POST /api/pos/submit-proof → { "success": true, "transaction": { "id", "transactionId", "amount", "currency", "status", "proofStatus", "createdAt" } } or { "success": false, "error", "recorded" }
